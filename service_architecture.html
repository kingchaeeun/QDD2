Quote Distortion Detection – Unified Architecture

Frontend (Chrome Extension) + Backend (FastAPI) + Shared Logic

Runtime Flow

  • Content Script injects overlay UI on Naver articles, extracts quotes via common/quote_extraction, and asks the background to analyze.
  • Background Service Worker calls http://127.0.0.1:8000/api/analyze with article context and quotes.
  • FastAPI Backend wraps the Python pipeline (app/pipeline.py), normalizes quotes via common/quote_extraction, and returns scored source candidates.
  • Overlay UI renders Qddfront’s panel/popup, highlights in-article quotes, and shows distortion breakdown.

Folder Map

extension/

  • background/ – service worker (API calls)
  • content_scripts/ – quote detection + overlay injector
  • overlay_ui/ – Qddfront React components
  • popup/ – action popup entry

backend/

  • main.py – FastAPI app with CORS + /api endpoints

common/

  • quote_extraction/ – shared JS (TS) + Python helpers

Data Contracts

  • Analyze request: { article: {title, reporter, paragraphs, url, text}, quotes: [{id, text, speaker}] }
  • Analyze response: { quotes: [{ id, text, speaker, sources: [{ id, title, sourceLink, originalText, distortionScore, similarityScore, scores: {...} }] }] }
